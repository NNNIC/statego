SETTING
Definition contents of 'setting' used for initialization
[setting]
Setting category.
For starter kit.
psgg
Specify a startup shortcut file (*. Psgg)
xls
Specify an Excel worksheet for storing data.
sub_src
Specify the user implementation file.
It is specified when the implementation file is different.
Note that this is used in pre release verion.
gen_src
Specify the file to insert the output.
manager_src
Specify the manager file.
manager_dir
Specify the management file folder.
Specify a relative path from the source folder.
template_src
Specify the source template.
It is used in the pre release version.
template_func
Specify a function template file.
help
Specify help file.
helpweb
Specify a html file for help to use in the help panel.
src_enc
Specify encoding of output source file.
The default is utf-8.
[setupinfo]
Setup information category
lang
Specify a programming language.
framework
Specify a framework name.
prefix
Specify a prefix.
This is used for the pre-release version.
xlsdir
Document storage folder at the state-machine creation.
This value is not referred to within the tool.
gendir
Output source file storage folder at the a state machine creation.
This value is not referred to within the tool.
genrdir
Output folder relative path: relative path from the document storage folder
incrdir
Include for macro files Relative path: Relative path from document folder path
code_output_start
Specify the output start mark.
code_output_end
Specify the output end mark.
[macro]
Macro category
[jpn]
Japanese explanation of the specified programming language setup.
[en]
English explanation of the specified programming language setup.
