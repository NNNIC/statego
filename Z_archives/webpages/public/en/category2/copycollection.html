Copy Collection

HOME サイトマップ

This function is designed to automatically remember the contents of a copy and reuse them.
To use this feature, you need to specify a location to record the copied content.

 

Specifying a working folder

 

To make a common working folder available to multiple StateGo's, place the working folder in the same or upper folder as the source.

 

Open the Copy Collection dialog

 

Click "CC" on the right end of the menu bar to open the Copy Collection dialog.

 

How to use the copy collection

 

The copied content is automatically added to the cache.
You can find it in the cache and use it by drag & drop.
It is also possible to move to the collection and use it from there as well by drag and drop.
