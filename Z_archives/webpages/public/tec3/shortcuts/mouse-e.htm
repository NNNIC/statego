Mouse operation list
Single mouse button mode
All mouse buttons have the same action.
Basic
Operation Description
Click ・ Focus the state directly under the pointer.
・ Display the context menu.
Hold button ・ Go to state transfer mode. (On state)
・ Go to slide mode. (On empty space)
・ Go to arrow mode. (On the branch point)
Pointer move + Hold button ・ Display the rectangular area for selection. (On empty space)
Auxiliary
Operation Description
Control key + Click ・ Turn ON / OFF the focus of the state directly under the pointer.
Shift key＋Hold button ・ Go to slide mode. (On empty space)
Control key + Hold button ・ Display the rectangular area for selection. (On empty space)
Multiple mouse button mode
An action is assigned to each mouse button.
Basic
　
Operation Description 　
Click Left Button ・ Focus the state directly under the pointer.
Hold Left Button ・ Go to slide mode. (On empty space)
Click Right Button ・ Display the context menu.
Hold Right Button ・ Go to state transfer mode. (On state)
・ Display the rectangular area for selection. (On empty space)
Auxiliary
Operation Description
Control Key + Click Right Button ・ Turn ON / OFF the focus of the state directly under the pointer.
