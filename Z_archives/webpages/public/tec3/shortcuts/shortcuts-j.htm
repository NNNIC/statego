ショートカット一覧
操作補助
キー 内容
Enter マウスポインタ直下のステートをフォーカスする。
またはコンテキストメニューを表示する。
左カーソル
右カーソル		 マウスポインタ直下のステートをフォーカスする。
または、遷移を移動する。
上カーソル
下カーソル		 ステート上の選択肢を移動する。 　 　
選択系 　
キー 内容 　
Home おおよそ左上のステートを一つ選択＆フォーカスする。 　
  End おおよそ右下のステートを一つ選択＆フォーカスする。
Ctrl + A 全ステートを選択する。
テンキー
テンキーの各キーの場所が表示エリアに相当する。
キー 内容
1 エリアの下段左を表示する。
2 エリアの下段中央を表示する。
3 エリアの下段右を表示する。
4 エリアの中段左を表示する。
5 エリアの中段中央を表示する。
6 エリアの中段右を表示する。
7 エリアの上段左を表示する。
8 エリアの上段中央を表示する。
9 エリアの上段右を表示する。
スクロール
Ctrl + カーソルキーでスクロールバーを操作する。
キー 内容
Ctrl + 左カーソル 表示エリアを左へスクロールする。
Ctrl + 右カーソル 表示エリアを右へスクロールする。
Ctrl + 上カーソル 表示エリアを上へスクロールする。
Ctrl + 下カーソル 表示エリアを下へスクロールする。
ズーム
キー 内容
< 縮小する。
> 拡大する。
? 100％を指定する。
コピー、貼り付け
キー 内容
Ctrl + C 選択したステートをコピーする。
Ctrl + V 選択したステートを張り付ける。
保存、変換
キー 内容
Ctrl + S 保存後変換する。
Ctrl + Shift + S 保存する。
フォルダ開く
キー 内容
Ctrl + 1 ドキュメントフォルダ開く。
Ctrl + 2 ソースフォルダ開く。
ソース表示
キー 内容
Ctrl + 3 ソース開く。
Ctrl + J ソース開く。
履歴操作
キー 内容
Ctrl + Z 変更履歴戻る。
Ctrl + Y 変更履歴進む。
[ フォーカス戻る。
] フォーカス進む。
Ctrl + H 変更履歴パネル表示・非表示。
Ctrl + T フォーカス履歴パネル表示・非表示。
検索
キー 内容
Ctrl + F 検索フォーム開く。
グループ
キー 内容
Ctrl + E 選択中グループに入る。
Ctrl + L グループから出る。
